	LCTP Latex Beamer Theme Readme
	
1) Copy the two '*.sty' files and the 'theme" directory to your lectures 
   main directory
   
2) Include the theme into your main tex file, after all 
   \usepackage statements with the following command
	
	\input{theme/theme}
	
3) The following latex statements HAVE to be included into your 
   main file
   
   \title{Linux Cluster in Theorie und Praxis}
   \subtitle{TITLE OF YOUR PRESENTATION}
   \date{DATE}
   \author{WELL NORMALLY YOUR NAME}
   \institute[ZIH TUD]{Zentrum f\"ur Informationsdienste und Hochleistungsrechnen -- TU Dresden}
   \room{YOUR ROOM NUMBER IF AVAILABLE}
	\address{STREET AND NUMBER}
	\city{ZIP CODE AND CITY}
   \phone{+49 0351 - 463 YOUR PHONE NUMBER}
	\email{YOUR EMAIL ADDRESS}

4) Make sure there is only this style included
